# 🚀 OOP Space Shooter (เกมยิงอวกาศ)

โปรเจกต์เกมยิงอวกาศ (Space Shooter) พัฒนาด้วยภาษา Python และไลบรารี Pygame สำหรับรายวิชา Object-Oriented Programming (OOP) 
โปรเจกต์นี้ประกอบด้วย Source Code ทั้งหมดสำหรับการทำงานของเกม

## 🛠️ ข้อกำหนดเบื้องต้น (Prerequisites)
ก่อนติดตั้งและรันเกม คุณจำเป็นต้องมีสิ่งเหล่านี้ในเครื่อง:
- **Python** เวอร์ชัน 3.8 ขึ้นไป (ทดสอบเพื่อให้แน่ใจว่าสามารถใช้พารามิเตอร์ของ Pygame ได้ครบถ้วน)
- **Git** (สำหรับการ Clone โปรเจกต์)

## 📥 วิธีการติดตั้ง (Installation)

โปรเจกต์นี้ได้เตรียมไฟล์สำหรับอำนวยความสะดวกในการติดตั้ง Library ไว้ 2 รูปแบบ คือ `requirements.txt` และ `pyproject.toml` คุณสามารถเลือกใช้ตามถนัด ดังนี้:

### ขั้นตอนเตรียมความพร้อม
1. เปิด Terminal (Command Prompt / PowerShell / Terminal)
2. เข้าสู่โฟลเดอร์ของโปรเจกต์
   ```bash
   cd path/to/OOP-Space-Shooter
   ```
3. *(แนะนำ)* สร้าง Virtual Environment เพื่อป้องกันไม่ให้ Library ไปตีกับโปรเจกต์อื่น
   ```bash
   python -m venv venv
   
   # บน Windows:
   venv\Scripts\activate
   
   # บน Mac/Linux:
   source venv/bin/activate
   ```

### วิธีที่ 1: ติดตั้งผ่าน `requirements.txt` (แนะนำ)
จัดการแพ็กเกจแบบมาตรฐาน สามารถรันคำสั่งด้านล่างเพื่อติดตั้ง `pygame` และแพ็กเกจที่จำเป็น:
```bash
pip install -r requirements.txt
```

### วิธีที่ 2: ติดตั้งผ่าน `pyproject.toml`
โปรเจกต์นี้รองรับมาตรฐานการติดตั้งแบบใหม่ คุณสามารถใช้งานคำสั่งนี้เพื่อติดตั้ง Library จากไฟล์ `pyproject.toml` ได้โดยตรง:
```bash
pip install .
```

## 🎮 วิธีการใช้งานและเริ่มเล่นเกม (How to Play)

เมื่อทำการติดตั้ง Library เสร็จสิ้นแล้ว สามารถเข้าเกมได้ดังนี้:

1. รันไฟล์โปรแกรมหลัก (Source Code เริ่มต้น)
   ```bash
   python main.py
   ```
2. เมื่อรันคำสั่ง หน้าต่างเกมจะปรากฏขึ้น

### 🎯 การควบคุม (Controls)
- `ปุ่มลูกศร (Arrow Keys)` หรือ `W, A, S, D`: ใช้สำหรับบังคับทิศทางของยานอวกาศ (ขึ้น, ลง, ซ้าย, ขวา)
- `Spacebar`: ใช้สำหรับยิงปืน / เลเซอร์
- `ESC`: ออกจากเกม

## 📂 โครงสร้างของโปรเจกต์ (Project Structure)
โปรเจกต์ประกอบด้วยไฟล์ Source Code ทั้งหมดดังนี้:
- `main.py`: ไฟล์โค้ดหลักสำหรับใช้รันเข้าสู่เกม
- `src/`: โฟลเดอร์รวมไฟล์โค้ด (Source Code) ส่วนต่าง ๆ ที่แยกตามหลักการ OOP (เช่น กลไกเกม, ยานผู้เล่น, ศัตรู)
- `assets/`: โฟลเดอร์สำหรับเก็บทรัพยากรที่ใช้ในเกม รูปภาพยาน, เสียงประกอบฉาก, เอฟเฟกต์
- `requirements.txt`: ไฟล์ประกาศรายการ Library (Dependencies) ที่จำเป็น
- `pyproject.toml`: ไฟล์ตั้งค่าและติดตั้งโปรเจกต์ Python ที่จัดไว้เป็นแพ็กเกจมาตรฐาน
